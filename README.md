# SMTP-Mailer
A C# smtp mailer windows application

[![N|Solid](https://linx.li/s/5d53u7ki.jpg)]


## Installation
```bash
git clone https://github.com/1onemike/SMTP-Mailer
```
## FEATURES
HTML messages with instant preview.

Multiple receivers.

Attachment option.

Delivery results.
